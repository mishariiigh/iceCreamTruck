✨ Features
•	Responsive design for mobile, tablet, and desktop
•	Modern UI with dessert-focused branding
•	Menu page showcasing products clearly
•	Contact page for customer inquiries
•	Fast performance using Next.js optimization
•	SEO-friendly structure
🛠️ Tech Stack
•	Next.js
•	React
•	TypeScript
•	Tailwind CSS

📦 Getting Started
Clone the repo:
git clone https://github.com/yourusername/ottawa-cool-treat.git

Install dependencies:
npm install

Run development server:
npm run dev

Open:
localhost	 

📁 Project Structure
•	/app – Pages and routing
•	/components – Reusable UI components
•	/public – Images and assets
•	/styles – Global styles
🎯 Purpose
This project was built to demonstrate modern front-end development skills, UI/UX design, and responsive web design for a real small business use case.
